React

Hello World
create-react-app에 의해 생성된 프로젝트에서 HelloWorld를 출력하자.
src폴더 하위에 components폴더를 생성하고 HelloWorld.js파일을 생성한 뒤 아래와 같이 작성한다. 

import React, { Component } from 'react';

class HelloWorld extends Component {
    render() {
        return (
            <div>
                <h1>HelloWorld</h1> 
            </div>
        );
    }
}

export default HelloWorld;
HelloWorld 컴포넌트를 출력하기 위해 App.js파일을 열어 아래와 같이 import하여 작성한다. 프로젝트를 실행하면 아래와 같이 HelloWorld가 출력된다.
