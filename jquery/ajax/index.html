jQuery.ajax()
  • 비동기 HTTP 요청을 수행한다.
ajax 작동예제

$.ajax({
	url: 'https://jsonplaceholder.typicode.com/posts',
	data: 'userId=1',
	dataType: "json",
	timeout: '1000',
	type: 'GET',
	success: function(data, status, xhr) {
		console.log(data);
	},
	error: function(xhr, status, thrown) {
		console.log(status);
	},
	complete: function(xhr, status) {
		console.log(status);
	}
});
Keyword Type Description
url String 요청을 보낼 주소
data PlainObject or String or Array 서버로 데이터를 전송한다. 전송시 쿼리스트링 형태로 변환한다.
dataType PlainObject or String or Array 서버로부터 받은 데이터의 형태를 지정한다.
(xml, html, script, json, jsonp, text)
timeout Number timeout에 설정된 시간이후 요청작업이 처리되지 않으면 실행을 취소한다.
type String(Default:GET) 데이터 전송 방식(GET, PUT, POST, DELETE...)
success Function
(Anything data, String textStatus, jqXHR jqXHR) 		요청작업이 성공했을때 실행한다.
error Function
(jqXHR jqXHR, String textStatus, String errorThrown) 		요청작업이 실패했을때 실행한다.
complete Function
(jqXHR jqXHR, String textStatus) 		요청작업이 종료 되었을때 실행한다.

Keyword Type Description
contentType PlainObject or String or Arra
(default: 'application/x-www-form-urlencoded; charset=UTF-8') 		데이터 전송시 serialize 된 형태로 변경되게 하지 않기 위해서 false로 지정한다.
processData Boolean(default:true) true일 경우 data옵션 값을 쿼리스트링으로 변환한다.
