jQuery.animate()
  • CSS속성 셋의 애니메이션을 실행한다.
jQuery.animate( properties [, duration ] [, easing ] [, complete ] ) 작동예제

$(function() {
	$( ".object" ).animate({
  		top: '+=100',
		left: '+=100'
	}, 1000, 'linear', function() {
		// Animation complete.
	});
});
Keyword Type Description
properties PlainObject 애니메이션 효과를 적용할 CSS 속성과 값의 객체
duration Number or String(default: 400) 애니메이션이 실행되는 시간을 지정
easing String(default: swing) 애니메이션의 시작, 중간, 마지막 부분의 속도 변화를 지정
complete Function() 애니메이션이 완료되었을 때 호출되는 함수
jQuery.animate( properties, options ) 작동예제

$(function() {
	$( ".object" ).animate({
		top: '+=100',
		left: '+=100'
	}, {
  	duration: 1000,
    easing: 'swing',
    complete: function() {
    	// Animation complete.
    }
  });
});
Keyword Type Description
properties PlainObject 애니메이션 효과를 적용할 CSS 속성과 값의 객체
options PlainObject 애니메이션 메소드에 추가적인 옵션의 맵
