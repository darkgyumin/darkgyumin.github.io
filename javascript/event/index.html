Event
  • 캡처링 : 부모 DOM -> 자식 DOM
  • 버블링 : 자식 DOM -> 부모 DOM
  • 하나의 DOM에 캡처링과 버블링을 같이 적용하면 캡처링 -> 버블링의 순서로 동작한다.
Parameter Description
event Required.
function Required.
useCapture Optional.
true - 캡처링, false(default) - 버블링 

element.addEventListener(event, function, useCapture)
element.removeEventListener(event, function, useCapture)
