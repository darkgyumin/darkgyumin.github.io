Built-in Object(내장 객체)
Keyword Description
Standard Built-in Object(Global Objects) 자바스크립트가 기본적으로 가지고 있는 객체들을 말한다.
내장객체 보기
BOM(Browser Object Model) 현재 브라우저의 창 또는 탭을 표현하는 객체이다.
최상위 객체는 window
DOM(Document Object Model) 현재 문서를 표현하는 객체이다.
최상위 객체는 document
