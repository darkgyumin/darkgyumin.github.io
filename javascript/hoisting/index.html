Hoisting
  • 변수가 사용된 후에 변수가 선언될수 있다.
  • 변수의 선언은 변수 사용후에 가능하지만 변수에 값을 할당하는것은 변수 사용전에 해야한다.

//변수가 선언되기 전에 사용하면 에러가 나야하지만 아래의 명령을 실행하면 undefined가 출력된다.
//variation 변수는 호이스팅에 의해 선언은 되었으나 값은 할당되지 않은 상태이다.
document.write(variation);

var variation = 5;

//변수에 값이 할당되어 정상적으로 출력된다.
document.write(variation);
