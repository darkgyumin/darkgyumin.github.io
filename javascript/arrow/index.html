Arrow Function
ES6이전의 익명함수를 간결하게 표기하며 자기 고유의 this가 없고 생성자로 사용할 수 없다.
기본문법

//매개변수가 없을 때
() => {}

//매개변수가 하나일 때
(param) => {}
param => {}

//매개변수가 여러개일 때
(param1, param2) => {}

//중괄호를 생략하면 값이 return된어 아래위 코드는 동일하게 동작한다.
param => {return param * 2}
param => param * 2
