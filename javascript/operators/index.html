Javascript

Operators
  • 프로그램의 논리식이나 산술식을 처리하기 위해 제공되는 다양한 기호들을 말한다.
산술연산자
Keyword Description
+ 값을 더한다.
- 값을 뺀다.
* 값을 곱한다.
/ 값을 나눈다.
% 나머지 값을 구한다.
++ 값이 증가한다.
-- 값이 감소한다. 

//산술연산자
console.log(1 + 2);	//3
console.log(1 - 2);	//-1
console.log(1 * 2);	//2
console.log(1 / 2);	//0.5
console.log(1 % 2);	//1

var i = 1;
i++;
console.log(i);	//2
i--;
console.log(i);	//1
대입연산자
Keyword Description
= 값을 대입한다.
+= 값을 더한 뒤 대입한다.
-= 값을 뺀 뒤 대입한다.
*= 값을 곱 뒤 대입한다.
/= 값을 나눈 뒤 대입한다.
%= 나머지 값을 대입한다. 

//대입연산자
var val = 10;
console.log(val);	//10
val += 10;
console.log(val);	//20
val -= 10;
console.log(val);	//10
val *= 2;
console.log(val);	//20
val /= 2;
console.log(val);	//10
val %= 2;
console.log(val);	//0
비교연산자
Keyword Description
== 값을 같은지 비교한다.
=== 값과 형이 같은지 비교한다.
!= 값이 다른지 비교한다.
!== 값과 형이 다른지 비교한다.
> 왼쪽값이 큰지 비교한다.
< 왼쪽값이 작은지 비교한다.
>= 왼쪽값이 큰거나 같은지 비교한다.
<= 왼쪽값이 작거나 같은지 비교한다. 

//비교연산자
console.log(1=='1');	//true
console.log(1==='1');	//false
console.log(1!='1');	//false
console.log(1!=='1');	//true
console.log(1>0);	//true
console.log(1<0);	//false
console.log(1>=0);	//true
console.log(1<=0);	//false
