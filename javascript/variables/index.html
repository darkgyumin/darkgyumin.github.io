Variables
Keyword Description
var 변수를 선언한다.
var를 이용한 변수 선언의 경우 함수레벨의 Scope를 가집니다.
let(ES6) let를 이용한 변수 선언의 경우 블록레벨의 Scope를 가집니다.
const(ES6) 변수를 상수값으로 선언한다.
변수 선언시 초기화 되어야 한다.
블록레벨의 Scope를 가집니다. 

var varname = value; //전역변수

function fnName() {
	var varname = value; //지역변수

	//함수내에서 var를 사용하지 않을 경우 전역변수가 된다.
	varname = value; //전역변수
}
