참조호출
  • 변수에 변수를 (=)대입하면 값 복사(call by value)로 대입이 되지만 (=&)대입하면 값 참조(call by reference)로 대입이 된다.

$valA = 'valueA Before';
$valB = $valA;
$valC =& $valA;

echo $valA;	//출력 valueA Before
echo $valB;	//출력 valueA Before
echo $valC;	//출력 valueA Before

$valA = 'valueA After';

echo $valA.'
';	//출력 valueA After
echo $valB.'
';	//출력 valueA Before
echo $valC.'
';	//출력 valueA After
