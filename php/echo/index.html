변수, 문자출력
변수
  • $변수명 = 값의 형태로 선언한다.
  • 변수명은 문자, 언더바(_)로 시작한다.
  • 영문의 대ㆍ소문자를 구분한다.

//$변수명 = 값;
$_variable = 'value';
$variable = 'value';
$Variable = 'value';
echo
  • 문자열을 화면에 출력한다.
  • 작은따옴표(')와 큰따옴표(")를 사용하여 문자열값을 표시한다.
  • 큰따옴표(")를 사용하면 중괄호({})로 둘러싸인 부분을 변수로 인식하여 값을 표시한다.

$_variable = '_value';

echo '변수값: ' . $_variable . '
';
echo '변수값: {$_variable}
';
echo "변수값: {$_variable}
";
sprintf
  • '%s'의 순서대로 인수를 문자열로 표시한다.

$variable = 'value';
$Variable = 'Value';

echo sprintf('변수출력 : %s, %s', $variable, $Variable);
Nowdoc 구문
  • <<<문자열 ...값... 문자열;의 형태로 사용된다.
  • 문자열은 터미네이션 ID라고 불린다.
  • 터미네이션 ID는 일반적으로 'EOF', 'EOL', 'EOT', 'END' 등을 사용한다.

$value = <<<EOL
변수에 대입하여 출력

개인 학습용 저장소입니다.

잘못된 내용이 있을수 있습니다.

EOL;

echo $value;

echo <<<EOF
직접 출력

개인 학습용 저장소입니다.

잘못된 내용이 있을수 있습니다.

EOF;
