환경설정

NDOE
https://nodejs.org
Node.js는 비동기 이벤트 기반의 JavaScript 런타임 환경이다.
Angular를 사용하기 위해서는 NPM(Node Package Manager)이 필요하다.
위의 링크에서 Node를 다운로드 받아 설치하면 NPM이 설치된다.
ANGULAR CLI 설치

//angular cli 설치하기
npm install -g @angular/cli

//angular cli 업데이트
ng update -g @angular/cli

//angular cli 삭제하기
ng uninstall -g @angular/cli
